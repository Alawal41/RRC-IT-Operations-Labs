# TrueNAS Active Directory Infrastructure

This project demonstrates the deployment of a centralized Active Directory lab integrated with TrueNAS for authentication and network file sharing. 

The environment includes Windows and Linux clients joined to the domain, SMB share configuration with permissions, Group Policy management, Raspberry Pi integration, and Ansible automation for software deployment. 

The lab was built in VirtualBox to simulate an enterprise IT infrastructure and strengthen skills in system administration, identity management, storage, and infrastructure automation.
